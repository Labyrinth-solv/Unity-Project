Cách cài đặt và chạy game

Sau khi tải FactoryGame_Release.zip, thực hiện các bước sau:

Giải nén file FactoryGame_Release.zip.
Mở thư mục vừa giải nén.
Chạy file:
FactoryGame.exe

Yêu cầu hệ thống

Game được build cho hệ điều hành Windows.

--------------->

Tài khoản đăng nhập: duc
Mật khẩu: 123456

--------------->

Cách điều khiển:

Đặt công trình: chuột trái
Xóa công trình: chuột phải
Xoay công trình: R
Xoay camera: Ctrl+R
Mở menu công trình: Q

Khuyến nghị:

Hệ điều hành: Windows 10 hoặc Windows 11
RAM: Tối thiểu 4GB
Dung lượng trống: Tối thiểu 500MB
Kết nối Internet: Cần thiết nếu sử dụng chức năng đăng nhập và bảng xếp hạng PlayFab
