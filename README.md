# Factory Game

Factory Game là game mô phỏng dây chuyền sản xuất được xây dựng bằng Unity. Người chơi đặt công trình, máy móc và băng chuyền để sản xuất vật phẩm theo mục tiêu của từng màn chơi.

## Tải game

Tải file game đã build:

```text
FactoryGame_Release.zip
```

## Cách cài đặt và chạy game

Sau khi tải `FactoryGame_Release.zip`, thực hiện các bước sau:

1. Giải nén file `FactoryGame_Release.zip`.
2. Mở thư mục vừa giải nén.
3. Chạy file:

```text
FactoryGame.exe
```

Lưu ý: Không xóa các file và thư mục đi kèm trong bản build, ví dụ:

```text
FactoryGame_Data
UnityPlayer.dll
MonoBleedingEdge
```

Các thành phần này cần thiết để game có thể chạy bình thường.

## Tài khoản đăng nhập demo

Người chơi có thể sử dụng tài khoản demo sau để đăng nhập:

| Tài khoản | Mật khẩu |
| --------- | -------- |
| `duc`     | `123456` |

## Cách điều khiển

| Chức năng          | Phím / thao tác |
| ------------------ | --------------- |
| Đặt công trình     | Chuột trái      |
| Xóa công trình     | Chuột phải      |
| Xoay công trình    | `R`             |
| Xoay camera        | `Ctrl + R`      |
| Mở menu công trình | `Q`             |

## Yêu cầu hệ thống

Game được build cho hệ điều hành Windows.

Khuyến nghị:

| Thành phần       | Yêu cầu                                                  |
| ---------------- | -------------------------------------------------------- |
| Hệ điều hành     | Windows 10 hoặc Windows 11                               |
| RAM              | Tối thiểu 4GB                                            |
| Dung lượng trống | Tối thiểu 500MB                                          |
| Kết nối Internet | Cần thiết nếu sử dụng đăng nhập và bảng xếp hạng PlayFab |

## Lưu ý

Nếu Windows hiển thị cảnh báo bảo mật khi mở file `.exe`, hãy chọn cho phép chạy nếu bạn tin tưởng nguồn tải.

Nếu game không mở được, hãy kiểm tra:

* Đã giải nén file `.zip` trước khi chạy chưa.
* Có đầy đủ thư mục `FactoryGame_Data` không.
* Có đầy đủ file `UnityPlayer.dll` không.
* Máy đang sử dụng hệ điều hành Windows phù hợp không.
* Kết nối Internet có hoạt động không nếu sử dụng đăng nhập hoặc leaderboard.
